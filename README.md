# IztroPromptGenerator

## 紫微斗数排盘提示语生成器

### 项目简介

- 本项目用于生成紫微斗数排盘的提示语
- 生成的提示语可以用于紫微斗数排盘的AI辅助
- 下一步：直接扔给DPSK，见证奇迹

本项目使用 [Angular CLI](https://github.com/angular/angular-cli) 版本 19.1.7 生成。

## 开发服务器

要启动本地开发服务器，请运行：

```bash
ng serve
```

服务器启动后，在浏览器中访问 `http://localhost:4200/`。当您修改任何源文件时，应用程序将自动重新加载。
